class cmoncrawl.aggregator.gateway_query.GatewayAggregator(urls: List[str], match_type: MatchType = MatchType.EXACT, cc_servers: List[str] | None = None, since: datetime = datetime.datetime(1, 1, 1, 0, 0), to: datetime = datetime.datetime(9999, 12, 31, 23, 59, 59, 999999), limit: int | None = None, max_retry: int = 5, prefetch_size: int = 3, sleep_base: float = 1.3, max_requests_per_second: int = 20)ÔÉÅ

This class is responsible for aggregating the index files from commoncrawl. It is an async context manager which can then be used as an async iterator which yields DomainRecord objects, found in the index files of commoncrawl.

It uses the commoncrawl index server to find the index files.

  • urls (List[str]) ‚Äì A list of urls to search for.

  • cc_indexes_server (str, optional) ‚Äì The commoncrawl index server to use. Defaults to ‚Äúhttp://index.commoncrawl.org/collinfo.json‚Äù.

  • match_type (MatchType, optional) ‚Äì Match type for cdx-api. Defaults to None.

  • cc_servers (List[str], optional) ‚Äì A list of commoncrawl servers to use. If None, then indexes will be retrieved from the cc_indexes_server. Defaults to None.

  • since (datetime, optional) ‚Äì The start date for the search. Defaults to datetime.min.
